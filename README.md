# 🎱 Snookr Web Game

A browser-based snooker game built with modern web technologies.
The goal of this project is to simulate realistic cue physics and provide an intuitive snooker experience.

---

## 🚀 Features

- 🎯 Drag-based cue control system
- 🎱 Realistic ball physics
- 🧠 Power control based on drag distance
- 🎮 Smooth aiming and rotation
- 🌐 Runs directly in the browser

---

## 🛠️ Tech Stack

- HTML5 / CSS3
- JavaScript / TypeScript
- React / Vite / Phaser 
- Canvas

---
